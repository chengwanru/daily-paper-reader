## 摘要
我们研究了以动作条件的世界建模作为一种可扩展的方式，用于学习机器人学习中可迁移的动力学先验。通过预训练一个模型来预测动作如何驱动视觉场景演变，所得到的世界模型能够捕捉超越外观级视频生成的可复用交互动力学。具体来说，我们在带有真实动作标注的大规模机器人操作数据上预训练了一个多视角交互基础扩散世界模型A2World。我们从两个互补的角度验证了学习到的动力学先验。首先，我们将A2World适配为任务或场景专用的真实世界模拟器A2World-sim，其长视界 rollout 通过用世界模型 rollout 替代真实机器人 rollout，支持基于模拟器的策略评估和可扩展的假设分析。其次，从相同的预训练权重出发，我们将A2World适配为视频-动作联合预测模型A2World-policy，该模型在视觉和指令条件下预测动作。在模拟基准和真实机器人设置上的实验表明，动作条件的世界模型预训练产生了可迁移的动力学先验，这对以模拟器为中心和以策略为中心的机器人学习都有益处。

## Abstract
We study action-conditioned world modeling as a scalable way to learn transferable dynamics priors for robot learning. By pretraining a model to predict how actions drive visual scene evolution, the resulting world model captures reusable interaction dynamics beyond appearance-level video generation. Concretely, we pretrain a multi-view interactive base diffusion world model, A2World, on large-scale robot manipulation data with real action annotations. We validate the learned dynamics priors from two complementary perspectives. First, we adapt A2World into a task- or scene-specialized real-world simulator, A2World-sim, whose long-horizon rollouts support simulator-based policy evaluation and scalable what-if analysis by replacing real-robot rollouts with world model rollouts. Second, starting from the same pretrained weights, we adapt A2World into a video-action joint prediction model, A2World-policy, that predicts actions under visual and instruction conditioning. Experiments across simulation benchmarks and real-robot settings demonstrate that action-conditioned world model pretraining yields transferable dynamics priors that benefit both simulator-centric and policy-centric robot learning.

## 2. 论文提出的方法论：核心思想、关键技术细节、公式或算法流程

### 核心思想
- **动作作为条件**：利用真实机器人动作标注作为监督，预训练一个多视图扩散世界模型A2World，使其学习动作到视觉演化的映射。
- **预训练→适配**：预训练后，将权重适配为两种变体：
  - **A2World-sim**：历史感知的自回归模拟器，用于长视界 rollout 和策略评估。
  - **A2World-policy**：MoE风格的视频-动作联合预测模型，作为指令条件策略。

### 关键技术细节
1. **A2World（基础世界模型）**
   - **输入**：条件帧 \(o_t\) 和未来动作块 \(a_{t+1:t+k}\)，输出未来观察 \(o_{t+1:t+k}\)。
   - **动作编码**：MLP将动作编码为嵌入，加到扩散时间步嵌入中，通过自适应层归一化（AdaLN）注入每个DiT块。
   - **多视图生成**：将多视图观察沿时间维度拼接，引入可学习的视图嵌入，并通过跨视图注意力模块强制视图间时空一致性。
   - **训练目标**：EDM去噪得分匹配损失，恢复干净隐变量。

2. **A2World-sim（长视界模拟器）**
   - **历史注入**：从历史序列中通过**姿态引导历史采样**（Algorithm 1）选择紧凑帧子集；将历史隐变量作为交叉注意力条件和自注意力键/值内存。
   - **自回归生成**：分块自回归滚动，采用**Self-forcing**训练策略（模型周期性在其自生成帧上训练）以提高长程稳定性。

3. **A2World-policy（视频-动作联合预测模型）**
   - **联合扩散公式**：对视频潜变量和动作潜变量独立加噪，使用共享基噪声水平按模态缩放。
   - **MoE-like块**：自注意力模块跨模态共享，但每模态保留独立的AdaLN和MLP分支。动作标记可关注视频和动作标记，视频标记通过自注意力更新。
   - **训练目标**：加权的联合去噪损失，包含视频和动作项。
   - **推理**：采用模态级CFG（分类器自由引导），可单独设置视频和动作的引导强度。

- **动作到视频预训练**产生**更强的可迁移先验**，优于文本条件或通用视频初始化。

## 7. 优点

- **可复用预训练范式**：单一预训练权重可适配为模拟器和策略，减少训练成本。
- **多视图一致性**：跨视图注意力确保时空一致生成，适应真实机器人多相机设置。
- **姿态引导历史采样**：高效压缩历史信息，提升长视界稳定性。
- **MoE-like联合扩散**：共享自注意力、独立分支，促进视频与动作的交互学习。
- **Self-forcing训练**：缓解自回归滚动中的训练-测试分布差异。
- **数据集一致批次策略**：避免多数据集冲突，允许在异质数据上稳定预训练。
- **丰富实验**：覆盖模拟、OOD、真实机器人，消融深入。
