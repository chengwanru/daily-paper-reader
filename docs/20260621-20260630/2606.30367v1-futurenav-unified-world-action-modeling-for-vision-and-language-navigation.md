## 摘要
连续环境下的视觉与语言导航要求智能体在自我中心观测中理解指令，同时维持长动作序列的空间理解。最近的导航基础模型通过扩展视觉语言模型取得了显著进展，但它们通常将导航主要学习为直接动作生成，而没有显式建模世界状态或预测其未来演变。我们提出了FutureNav，一种基于VLM的统一世界-动作建模框架，用于视觉与语言导航。具体而言，FutureNav联合编码文本、视觉和空间特征，并将其输入到LLM中，同时优化四个目标以实现世界与动作的联合建模：用于导航动作预测的动作策略目标、用于建模状态转移的逆和正动力学目标，以及用于预测未来空间状态的未来生成目标。这种统一架构增强了动作预测，同时显式建模世界，且不牺牲推理速度。大量实验表明，仅使用4B规模的骨干网络，FutureNav在多个VLN基准测试上取得了最先进的性能，并大幅优于先前的VLN方法，为未来VLN的世界-动作模型铺平了道路。我们将发布代码和模型以支持未来研究。

## Abstract
Vision-and-language navigation (VLN) in continuous environments requires an agent to ground instructions in egocentric observations while maintaining spatial understanding across long action sequences. Recent navigation foundation models have shown strong progress by scaling vision-language models, but they often learn navigation primarily as direct action generation, without explicitly modeling world states or predicting their future evolution. We introduce FutureNav, a VLM-based unified world-action modeling framework for vision-and-language navigation. Specifically, FutureNav jointly encodes text, visual, and spatial features and feeds them into the LLM, and optimizes four objectives for simultaneous world and action modeling: an action policy objective for navigation action prediction, inverse and forward dynamics objectives for modeling state transitions, and a future generation objective for predicting future spatial states. This unified architecture strengthens action prediction while explicitly modeling the world, without sacrificing inference speed. Extensive experiments show that, with only a 4B-scale backbone, FutureNav achieves state-of-the-art performance on multiple VLN benchmarks and substantially outperforms prior VLN methods, paving the way toward future world-action models for VLN. We will release the code and models to support future research.

**核心问题**：  
连续环境下的视觉与语言导航（VLN）要求智能体根据自然语言指令，在未知3D环境中通过自我中心观测执行长序列动作。现有VLN方法主要依赖直接的动作生成（观测→动作映射），缺乏对**世界状态**（当前空间布局、物体位置等）以及**动作如何改变世界状态**的显式建模，导致长程空间推理能力不足、决策鲁棒性差。

**背景**：  
- 早期VLN方法使用路径点预测、拓扑规划、语义地图等显式导航结构。  
- 近年来基于VLM的方法（如NaVid、JanusVLN）将导航建模为观测到动作的直接生成，但未利用状态转移与未来预测。  
- 同期世界模型方法（如NWM、NavMorph）尝试预测未来观测或隐式状态，但通常将策略学习与世界建模分离，或在推理时引入额外计算开销。

**整体含义**：  
FutureNav旨在将**世界建模**与**动作预测**统一到单个VLM框架中，通过联合学习动作策略、状态转移（前向/逆动态）、未来状态生成，提升导航性能，同时保持推理效率。

## 2. 论文提出的方法论

**核心思想**：  
构建一个基于VLM的统一世界-动作建模框架，在训练时使用四个互补目标联合优化，使模型不仅学会选择动作，还学会理解当前世界状态及其变化。

**关键技术细节**：  
- **输入编码**：  
  - 文本与视觉：使用VLM backbone（Qwen3-VL-4B/8B）对指令和观测历史进行编码。  
  - 空间编码：使用**冻结的空间编码器VGGT**提取每帧的空间特征（几何感知），并通过可训练的合并器（两层MLP）映射到LLM隐空间，再以残差加权的形式融合到视觉token中（权重α=0.2）。  
- **四个训练目标**：  
  - **动作策略（Action Policy）**：标准自回归动作解码（STOP/MOVE_FORWARD/TURN_LEFT/TURN_RIGHT），使用交叉熵损失。  
  - **前向动态（Forward Dynamics）**：基于当前观测隐状态和教师强制下的真实动作隐状态，预测下一观测的空间特征（MSE损失）。  
  - **逆动态（Inverse Dynamics）**：基于两相邻观测的隐状态及其差值，推断中间执行的动作（分类交叉熵）。  
  - **未来生成（Future Generation）**：不使用动作token，仅基于当前观测隐状态预测下一观测空间特征（MSE损失），提供动作无关的短期演化先验。  
- **训练与推理分离**：  
  训练时四个头共同优化，推理时仅保留动作策略分支，不引入额外计算。

总损失 L = L_policy + λ_f L_forward + λ_i L_inverse + λ_g L_generation，其中λ设为0.1。
