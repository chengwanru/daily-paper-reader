## 摘要
世界-动作模型通过联合预测未来视觉状态和动作，在机器人操作中展现了良好的性能。然而，现有方法主要依赖短期历史与短时域未来预测，这对于长期任务而言是不足的，因为任务的正确执行依赖于早期观测和任务进展。这类时间依赖任务需要有效利用互补的时间信息，包括近期局部上下文、跨阶段历史事件、即时未来动态以及全局任务进展。为解决长期遗忘与全局任务状态感知不足的问题，我们提出了DiM-WAM，一种记忆增强的世界-动作模型，它融合了多尺度历史上下文、局部未来动态和全局任务进展。该记忆从真实观测中提取紧凑的视觉事件信息，通过基于相似性的独立合并更新多个记忆库，然后读取带有库标识和时间嵌入的长期上下文，以条件化视频和动作去噪。一个进度监督目标进一步促使记忆令牌不仅编码已完成的历史事件，还编码当前任务阶段及其对剩余任务的影响。在RMBench上，DiM-WAM将平均成功率从LingBot-VA的28.4%提升至69.8%，超过了显式记忆的Mem-0基线（42.0%）。在四个真实Franka任务中，它将平均阶段成功率从70.7%提升至91.5%，完整任务成功率从52.5%提升至80.0%。项目页面：https://wangkai-casia.github.io/dim-wam/。

## Abstract
World-action models have shown promising robot-manipulation performance by jointly predicting future visual states and actions. However, existing methods mainly rely on short-term history and short-horizon future prediction, which is insufficient for long-horizon tasks whose correct execution depends on earlier observations and task progress. Such temporally dependent tasks require effective use of complementary temporal information, including recent local context, cross-stage historical events, immediate future dynamics, and global task progress. To address long-term forgetting and poor awareness of the global task state, we introduce DiM-WAM, a memory-augmented world-action model that integrates multi-scale historical context, local future dynamics, and global task progress. The memory extracts compact visual event information from real observations, updates multiple memory banks through independent similarity-based merging, and then reads the bank-identity- and time-embedded long-term context to condition video and action denoising. A progress-supervision objective further encourages memory tokens to encode not only completed historical events but also the current task stage and its implications for the remaining task. On RMBench, DiM-WAM raises average success from 28.4% with LingBot-VA to 69.8%, exceeding the explicit-memory Mem-0 baseline at 42.0%. On four real-world Franka tasks, it improves average stage success from 70.7% to 91.5% and full-task success from 52.5% to 80.0%. # 论文深度总结：DIM-WAM：基于多样化历史事件记忆的世界-动作模型

## 1. 论文的核心问题与整体含义（研究动机和背景）

*   **研究背景与问题：** 现有的世界-动作模型（WAMs）在机器人操作任务中表现良好，但其核心机制主要依赖**短期历史**和**短时域未来预测**。然而，许多复杂的长期操作任务（如多步骤整理、组装等）具有**非马尔可夫性质**，其成功执行高度依赖于**早期观测**和**当前的任务进度**。
*   **核心挑战：** 长期任务的成功需要模型处理四种互补的时间信息类型：
    1.  **短期历史：** 刚发生的事件（局部上下文）。
    2.  **长期历史：** 已完成子任务和早期状态（跨阶段事件）。
    3.  **短期未来：** 即将发生的视觉变化和动作结果（局部动态）。
    4.  **全局任务进度：** 当前状态在整个任务中的位置（全局感知）。
*   **现有方法的不足：**
    *   简单增加历史窗口长度会导致计算量激增，且早期关键信息可能被注意力机制“稀释”。
    *   若将生成的预测状态纳入长期上下文，预测误差会递归累积，导致模型内部状态逐渐偏离真实环境。
    *   现有记忆增强的VLA方法主要针对**动作预测**，而WAM中的记忆还需直接参与**未来视觉状态的生成**，二者对记忆的需求不同。
*   **本文动机：** 为了解决长期遗忘和全局任务状态感知不足的问题，本文提出一种**记忆增强的世界-动作模型（DIM-WAM）**，旨在显式地、结构化地管理长期任务中的多种时间信息。

## 2. 论文提出的方法论：核心思想、关键技术细节

*   **核心思想：** 将长期决策所需的信息显式分解为“短期历史、跨阶段历史、短期未来、全局进度”四类，并设计一个**多库历史事件记忆**系统来组织和压缩跨阶段历史，同时通过**任务进度监督**来塑造记忆的全局感知能力。
*   **关键技术细节（DIM-WAM框架）：**

    1.  **多尺度世界-动作建模：**
        *   模型`fθ`在给定语言指令`c`、局部上下文`Ci`和长期记忆`Mi`的条件下，联合预测未来视觉潜变量`ẑ`和动作序列`â`。
        *   局部上下文`Ci`处理短期动态，而长期记忆`Mi`补充跨阶段证据，解决视觉歧义。

    2.  **多类型历史事件记忆（Multi-Bank Temporal Memory）：**
        *   **多记忆库结构（Why multiple banks?）：** 设计K个并行记忆库，而不是一个单一的库。这避免了不同类型的历史信息（如初始位置、关键交互结果）在单一容量内竞争资源。各库通过独立压缩和学习，形成互补的历史视图。
        *   **压缩视觉事件提取：** 每次交互后，模型从真实观测中提取紧凑的**视觉事件令牌** `ũi`。
        *   **独立有界压缩（更新规则）：** 每个记忆库独立维护一个固定长度的事件序列。当容量满时，通过一个**合并优先级函数** `ρ(u,v)`（结合语义相似度和时间距离）来评估并合并最冗余的相邻历史事件。合并后的结果通过累计质量加权得到。更新规则的核心是：**只有当新事件比合并最冗余历史对更能提供新信息时，才被写入**。
        *   **算法流程（Algorithm 1）：** 详细描述了每个库的更新步骤，包括初始化、处理重复项、寻找最冗余对、以及决定是否合并和写入新事件。

    3.  **协作式记忆读取（Cooperative Memory Reading）：**
        *   在预测前，为每个库的令牌添加**库标识嵌入** `bk`，然后按时间顺序展平所有令牌，并添加**旋转位置编码（RoPE）** 时间嵌入，形成最终的记忆条件`Ri`。
        *   该`Ri`同时作为**视频去噪**和**动作去噪**的条件，确保两个分支基于相同的长期任务状态进行预测，防止产生不一致的阶段估计。

    4.  **任务进度感知记忆学习（Task-Progress-Aware Memory Learning）：**
        *   设计一个辅助任务：从聚合的长期记忆`Ri`中预测当前段的**离散任务进度分布** `pi`。
        *   通过交叉熵损失`Lprog`进行监督，迫使长期记忆不仅编码“发生了什么”，还要编码“当前处于任务的哪个阶段”，从而提升全局状态感知能力。

    5.  **训练目标：**
        *   总损失函数为：`L = L_video + λ_a * L_action + λ_div * L_div + λ_prog * L_prog`。
        *   除了视频/动作损失和进度损失外，还引入**多样性损失** `L_div`，通过最小化不同记忆库平均特征之间的点积平方，鼓励各个库学习互补的、非冗余的信息，防止多库结构坍塌。

## 6. 论文的主要结论与发现

*   **核心结论：** DIM-WAM通过结构化、多样化的记忆管理，有效解决了长期操作任务中的长期遗忘和全局感知问题，显著提升了任务成功率。
