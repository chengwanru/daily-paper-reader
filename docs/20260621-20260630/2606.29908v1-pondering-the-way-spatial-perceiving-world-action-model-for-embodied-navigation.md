## 摘要
现有的基于世界模型的视觉导航规划器通常遵循以验证为核心的范式，将目标意图与轨迹合成解耦。这种方法存在候选依赖、计算开销大以及采样动作与预测视觉不一致等问题。为了解决这些问题，我们提出了SWAM（空间感知世界动作模型），一个以任务为中心的联合观察-动作生成框架。给定起始和目标RGB观测，SWAM通过单次推理同时生成中间RGB-D序列和相应的动作轨迹，促进目标一致的轨迹生成并提高空间可行性。虽然SWAM在训练期间利用深度伪标签内化空间先验，但在推理时仅需单目RGB输入。我们进一步引入了视觉引导的动作精炼模块和轨迹尺度正则化损失，以强制运动与视觉线索之间的细粒度对齐，同时稳定不同距离的预测。大量实验表明，SWAM在成功率、轨迹精度和推理效率方面显著优于最先进的两阶段规划器，并在未见环境中展现出强大的零样本泛化能力。

## Abstract
Existing world model-based planners for visual navigation typically follow a verification-centric paradigm, decoupling goal intent from trajectory synthesis. This approach suffers from candidate dependence, heavy computational overhead, and inconsistencies between sampled actions and predicted visuals. To address these issues, we propose SWAM (Spatial-perceiving World Action Model), a task-centric joint observation-action generation framework. Given start and goal RGB observations, SWAM performs single-pass inference to simultaneously generate intermediate RGB-D sequences and corresponding action trajectories, promoting goal-consistent trajectory generation and improved spatial feasibility. While SWAM leverages depth pseudo-labels during training to internalize spatial priors, it requires only monocular RGB input at inference time. We further introduce a visual-guided action refinement module and a trajectory-scale regularization loss to enforce fine-grained alignment between motion and visual cues while stabilizing predictions across varying distances. Extensive experiments show that SWAM significantly outperforms state-of-the-art two-stage planners in success rate, trajectory accuracy, and inference efficiency, while demonstrating robust zero-shot generalization to unseen environments.

## 2. 方法论：核心思想与关键技术细节

### 核心思想
SWAM（Spatial-perceiving World Action Model）是一个**联合观测-动作生成模型**。给定起始 RGB 观测和目标 RGB 观测，模型通过**单次前向推理**同时生成：
- 中间 RGB-D 观测序列（N 步）
- 对应的动作轨迹（平面位移 Δx, Δy）

训练时利用深度伪标签（DepthAnything v3）提供空间先验，推理时仅需单目 RGB。联合生成过程确保动作与视觉演化相互约束，实现目标对齐、时间一致性和空间可行性。

### 关键技术细节
1. **潜在表示与分词化**：
   - 每个观测（RGB + 估计深度）分别用冻结的 3D VAE（CogVideoX 的 VAE）编码为潜在特征图。
   - 动作序列通过可学习 MLP 投影为相同维度的 token 矩阵，形成统一的 token 序列。

2. **多模态序列构建**：
   - 将中间 RGB 潜在、深度潜在、动作 token 拼接为扩散目标序列 X₀。
   - 将起始与目标的 RGB/深度潜在作为条件 c。

3. **去噪过程**：
   - 采用 DDPM 风格的扩散 Transformer（DiT），迭代去噪 T 步，预测噪声残差。公式：
     ```
     X_{t-1} = (1/√α_t) * (X_t - (1-α_t)/√(1-α̅_t) * ε_θ(X_t, c, t)) + σ_t z
     ```
   - 去噪完成后，RGB/深度潜在用 VAE 解码，动作 token 通过轻量网络配合 VGAR 模块解码为平面动作。

4. **视觉引导动作精化模块（VGAR）**：
   - 在动作解码前，利用生成的 RGB-D 潜在通过交叉注意力精化动作 token，引入门控残差机制控制视觉信息流入：
     - C = CrossAttn(LN(X_a), LN(X_v), LN(X_v))
     - ΔX_a = G ⊙ W(C), G = σ(MLP([LN(X_a); C]))
     - X'_a = X_a + ΔX_a

5. **轨迹尺度正则化损失（TSR）**：
   - 直接监督去噪后动作积分得到的终点位置与真实目标位置的偏差，公式：
     ```
     L_TSR = ||ŝ_N - p_G||² / N,  其中 ŝ_N = Σ_{n=1}^N â_n
     ```
   - 可避免单步动作误差累积导致的全局漂移。

6. **联合训练目标**：
   - L = L_DDPM + λ_TSR * L_TSR
   - L_DDPM 为标准扩散去噪损失，L_TSR 为轨迹正则化。

7. **模型初始化**：
   - 基于预训练的 CogVideoX 视频扩散模型，新加模块（如深度编码、动作头、VGAR）全部初始化为零权重，以保持初始行为与预训练一致，逐步学习耦合。

8. **训练策略细节**（来自附录）：
   - 固定序列长度训练（8 对观测-动作，共 9 帧）。
   - 还训练了变长版本（N ∈ {9,17,33,65}）用于长距离可视化，但主实验均使用固定长度。
   - 使用 bf16 混合精度、梯度检查点、batch size 1、Adam 优化器（lr=1e-4，warm-up 1000 步，梯度裁剪 max norm=1.0）。

## 8. 不足与局限

- **失败场景**：
