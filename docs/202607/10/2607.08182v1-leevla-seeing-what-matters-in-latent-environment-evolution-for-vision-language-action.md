## 摘要
视觉-语言-动作（VLA）模型旨在将多模态输入映射到机器人动作。然而，大多数现有方法由于均匀处理所有视觉标记并使用人为选择的因素进行推理，缺乏强调任务关键证据的机制且忽略潜在因素，因此难以覆盖复杂的动态场景。为解决这一问题，我们提出LEEVLA，一种在潜在环境演化中关注关键信息的VLA架构，该架构显式引导模型关注信息丰富区域，同时保留潜在世界表示的结构化演化。为识别显著且与指令相关的区域，我们引入漂移引导动态优先级（DGDP），它结合动态位置优先级（DPP）与语义漂移引导（SDG），在训练过程中引导VLA智能体关注何处。在此基础上，我们引入结构化特征流生成（SFFG），通过原型到外围（P2P）预测建模这些优先级特征如何在潜在空间中演化，并引入互邻域对比（MC）损失以维持邻域间的拓扑一致性。DGDP与SFFG共同构成一个任务感知的“何处-如何”训练框架。在VLA基准上的大量实验表明，LEEVLA持续优于先前方法，证实了显式任务证据引导和结构化潜在推理对于可扩展VLA都至关重要。我们的代码可在https://github.com/LyuQi127/LEEVLA获取。

## Abstract
Vision-language-action (VLA) models aim to map multimodal inputs to robot actions. However, most existing approaches struggle to cover complex dynamic scenarios due to treating all visual tokens uniformly and reasoning with human-selected factors, which lack mechanisms to emphasize task-critical evidence and ignore underlying factors. To address this issue, we propose LEEVLA, a VLA architecture for seeing what matters in Latent Environment Evolution that explicitly guides the model toward informative regions while preserving the structured evolution of latent world representations. To identify salient and instruction-relevant regions, we introduce drift-guided dynamic prioritization (DGDP), which combines dynamic position prioritization (DPP) with semantic drift guidance (SDG) to guide the VLA agent where to attend during training. On top of this, we introduce structured feature flow generation (SFFG), which models how these prioritized features should evolve in latent space via prototype-to-periphery (P2P) prediction, and a mutual-neighborhood contrastive (MC) loss to maintain topological consistency among neighborhoods. Together, DGDP and SFFG form a task-aware "where-how" training framework. Extensive experiments on VLA benchmarks show that LEEVLA consistently outperforms prior methods, confirming that explicit task-evidence guidance and structured latent reasoning are both crucial for scalable VLA. Our code is available at https://github.com/LyuQi127/LEEVLA.

## 2. 方法论

### 核心思想
构建一个任务感知的“where-how”训练框架：
- **DGDP（Drift-Guided Dynamic Prioritization）**：自动发现需要关注的区域（where）；
- **SFFG（Structured Feature Flow Generation）**：建模这些区域在潜在空间中的结构化演化（how）。

### 关键技术细节

#### 2.1 漂移引导动态优先级（DGDP）
- **动态位置优先级（DPP）**：基于相邻时间步特征余弦相似度的变化计算每个 patch 的“动态性分数”θᵢ = 1 - cos(v_{t,i}, v_{t+T,i})，从而突出与交互相关的活跃区域。
- **语义漂移引导（SDG）**：计算每个 patch 的指令相关性得分 r_{t,i} = max_j ⟨\tilde{x}_{t,i}, \tilde{c}_j⟩，并定义语义漂移 Δᵢ = clip((r_{t+T,i} - r_{t,i})/τ, -δ, δ)，归一化后得到语义漂移引导因子 ωᵢ = exp(\tilde{Δᵢ}/τ)。最终优先级权重 βᵢ = σ(ωᵢ · θᵢ)（σ 为 sigmoid）。

#### 2.2 结构化特征流生成（SFFG）
- **原型到外围预测（P2P）**：对未来特征进行聚类（K-means），在每个簇内按特征到质心的欧氏距离从近到远排序，然后按此顺序施加加权余弦嵌入损失，损失函数为：
  LP2P = (1/|C|) Σ_{ℓ=1}^{|C|} (1/|\overrightarrow{c_ℓ}|) Σ_{j∈\overrightarrow{c_ℓ}} (α + β_j) φ(\hat{v}_{t+T,j}, v_{t+T,j})
  其中 α 为全局调制因子（保持对所有区域的弱关注），β_j 为上述优先级权重。
- **互邻域对比损失（MC Loss）**：基于二阶 k-NN 互邻域构建高置信度正样本对，采用 InfoNCE 损失 LMC = -(1/|I|) Σ_{i∈I} (1/|G_i^+|) Σ_{j∈G_i^+} log [exp(S_{ij}/τ_c) / Σ_{ℓ≠i} exp(S_{iℓ}/τ_c)]，其中 G_i^+ 为与 anchor i 互为邻域的 token 集合。该损失抑制噪声聚类带来的非对称连接。

#### 2.3 训练目标
总损失：L_total = λ₁ L_action + λ₂ LP2P + λ₃ LMC
- L_action：动作预测的 L1 回归损失（并行预测连续动作 chunk）。
- DGDP 和 SFFG 仅在训练时使用，推理阶段无额外开销。

## 3. 实验设计

### 数据集与场景
- **LIBERO 基准**：包含四类任务——Spatial（空间）、Object（物体）、Goal（目标）、Long（长序列）。每个任务有 10 条语言指令，评估 50 个 episode，3 个随机种子。
- **CALVIN 基准**：评估长序列任务的成功率（task 1 到 5）及平均任务长度（5 分制）。
- **真实世界实验**：使用 UR5 机械臂，完成三个任务——放置物体、按按钮、关闭抽屉，每个任务 20 次连续试验。

### 对比方法
- **小规模模型（<1B 参数）**：Octo (0.1B)、UniACT (0.5B)、Seer (0.3B)、DreamVLA (0.3B)、FLOWER (1B)。
- **大规模模型（≥3B 参数）**：OpenVLA (7B)、π0 (3B)、π0.5 (3B)、OpenVLA-OFT (7B)、UniVLA (7B)、CoT-VLA (7B)、MemoryVLA。
- 主要指标：平均成功率（%）。

### 实验结果
- **LIBERO**：LEEVLA-mini (0.5B) 平均 97.5%（优于 FLOWER 的 95.7%）；LEEVLA-large (7B) 平均 98.2%（优于 π0.5 的 97.5% 和 OpenVLA-OFT 的 97.1%）。

- **真实世界**：LEEVLA-large 平均成功率 78.5%（OpenVLA 为 40%）。
